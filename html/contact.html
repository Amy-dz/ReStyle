FAQ

Pour acheter un article, connectez-vous à votre compte, explorez les annonces, et cliquez sur "Acheter" pour l’article qui vous intéresse. Vous serez redirigé vers une page sécurisée pour finaliser votre paiement.

Rendez-vous dans votre profil, accédez à la section "Mes annonces" et sélectionnez l'annonce que vous souhaitez modifier ou supprimer. Utilisez les boutons correspondants pour effectuer vos changements.

Si vous n’avez pas reçu votre article dans les délais estimés, contactez d'abord le vendeur via la messagerie intégrée. Si le problème persiste, contactez notre équipe via le formulaire de contact pour ouvrir un litige.

Toutes les transactions effectuées sur Restyle sont sécurisées par un système de paiement tiers de confiance. Vos informations bancaires sont cryptées et ne sont jamais partagées avec le vendeur.

Si vous remarquez une annonce ou un comportement suspect, utilisez le bouton "Signaler" disponible sur la page de l’annonce ou dans le profil de l'utilisateur concerné. Vous pouvez également nous contacter directement via le formulaire de contact.

Les annulations ou remboursements sont possibles dans certaines conditions. Consultez nos Conditions Générales d'Utilisation pour en savoir plus ou contactez notre équipe via le formulaire pour examiner votre situation.
